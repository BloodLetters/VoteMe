<h2>VoteMe</h2>

VoteMe is a <a href="https://github.com/vexsoftware/votifier">Votifier</a>  port of the Bukkit plugin for <a href="https://pumpkinmc.org/">PumpkinMC</a>. It listens for incoming vote events from popular server listing websites and makes those votes available to your PumpkinMC server so you can reward players, trigger commands, and build vote-based features.
</div>

## Key Features
- **Votifier-compatible receiver**: Accepts vote payloads from services that support the Votifier protocol.
- **PumpkinMC integration**: Designed specifically to fit PumpkinMC plugin workflows.
- **Cross-environment friendly**: Works across a variety of operating systems
